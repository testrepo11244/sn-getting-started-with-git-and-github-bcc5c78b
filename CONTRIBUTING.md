# Contributing to the Simple Interest Calculator

We welcome contributions of all kinds! Whether you have a bug report, a bug fix, documentation improvements, enhancements, or new ideas, your help is appreciated.

## How to Contribute

1. **Fork the repository** and create a new branch for your work.
2. **Make your changes** and ensure that the project still works as expected.
3. **Write clear commit messages** describing what you have done.
4. **Open a Pull Request** targeting the `main` branch of the original repository.
5. **Participate in the review process** – we may ask for clarifications or adjustments.

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

Thank you for helping make this project better!
